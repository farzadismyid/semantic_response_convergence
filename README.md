# Semantic Response Convergence

A modular research framework for studying whether LLM agents compress prompt variation into semantically similar responses.

Initial conditions:

- A1: Baseline LLM
- A2: System Prompt
- A3: RAG
- A4: System Prompt + RAG
- A5: System Prompt + RAG + Memory

Initial prompt dataset:

- Bitext customer support dataset

Initial retrieval corpus:

- Separate handcrafted customer-support knowledge base
